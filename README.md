# Data Visualization on IPL Dataset

This project performs **data visualization and analysis on the IPL Matches dataset (2008–2022)** using Python.

## Tools Used

* Python
* NumPy
* Pandas
* Matplotlib
* Seaborn
* Jupyter Notebook

## Tasks Performed

* Total number of matches played
* Total IPL seasons
* Matches played in each season
* Maximum and minimum matches in a season
* Total matches won by each team
* Matches played in each stadium
* Analysis of toss winner vs match winner
* Top 15 Player of the Match winners
